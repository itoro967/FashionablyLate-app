## 環境構築

### Dockerビルド
1. ```git clone <リンク>```
1. ```sudo docker-compose up -d```

### Laravel環境構築
1. ```sudo docker-compose exec php bash```
1. ```composer install```
1. ```cp .env.example .env```
1. ```php artisan key:generate```
1. ```php artisan migrate```
